class ra2ce.network.networks.Network(network_config: NetworkConfigData, graph_files: GraphFilesCollection)[source]#

Bases: object

Network in GeoDataFrame or NetworkX format.

Networks can be created from shapefiles, OSM PBF files, can be downloaded from OSM online or can be loaded from feather or gpickle files. Origin-destination nodes can be added.

Attributes:

config: An object with the configuration details on how to create and adjust the network.
